# Dataset de Figuras Geométricas

Este repositorio contiene un dataset de imágenes organizado para tareas de **clasificación de figuras geométricas** en visión por computadora y aprendizaje automático.

## Clases

El dataset incluye:

- Círculo
- Cuadrado
- Rectángulo
- Triángulo

## Estructura del dataset

El contenido está dividido en tres subconjuntos:

- `train`: conjunto de entrenamiento
- `val`: conjunto de validación
- `test`: conjunto de prueba
